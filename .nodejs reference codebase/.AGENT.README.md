# Auto-AI Agent - Strategy Game Automation

A comprehensive browser automation framework for autonomous strategy game play using vision-based AI agents.

## Overview

This implementation adds vision-based autonomous AI agent capabilities to the Auto-AI framework, specifically designed for slow-paced web-based strategy games.

## Features

### Phase 1: Core Action Enhancements

| Function                            | Description                     | Usage                                               |
| ----------------------------------- | ------------------------------- | --------------------------------------------------- |
| `api.drag(source, target, options)` | Drag with configurable duration | `api.drag('.unit', '.target', { durationMs: 500 })` |
| `api.clickAt(x, y, options)`        | Raw coordinate clicking         | `api.clickAt(450, 320)`                             |
| `api.multiSelect(items, options)`   | Shift/Ctrl multi-select         | `api.multiSelect([1, 3, 5], { mode: 'add' })`       |
| `api.press(key, modifiers)`         | Keyboard with modifiers         | `api.press('w', ['ctrl'])`                          |
| `api.typeText(text)`                | Human-like typing               | `api.typeText('hello')`                             |
| `api.hold(key, durationMs)`         | Hold key down                   | `api.hold('w', 500)`                                |

### Phase 2: Vision-Enhanced Agent

**Game Agent** - Autonomous agent with verification:

```javascript
await api.gameAgent.run('Build barracks and train 5 footmen', {
    maxSteps: 30,
    stepDelay: 500,
    verifyAction: true,
    stuckDetection: true,
    maxAttemptsWithoutChange: 5,
});
```

**Supported LLM Actions:**

- `{ "action": "click", "selector": "..." }`
- `{ "action": "clickAt", "x": 450, "y": 320 }`
- `{ "action": "drag", "source": {...}, "target": {...}, "duration": 500 }`
- `{ "action": "multiSelect", "items": [1, 2, 3], "mode": "add" }`
- `{ "action": "type", "selector": "...", "value": "..." }`
- `{ "action": "press", "key": "Enter" }`
- `{ "action": "scroll", "value": "down" }`
- `{ "action": "wait", "value": 5000 }`
- `{ "action": "done" }`

### Phase 3: Strategy Game Primitives

#### Unit Selection (`api.game.units.*`)

```javascript
await api.game.units.selectUnit('.unit-1'); // Select single unit
await api.game.units.selectByArea({ x: 100, y: 100 }, { x: 300, y: 300 }); // Box select
await api.game.units.selectAll(); // Ctrl+A
await api.game.units.deselectAll(); // Click empty area
await api.game.units.issueCommand({ x: 500, y: 400 }, 'attack'); // Right-click command
await api.game.units.selectGroup(1); // Select group 1-9
await api.game.units.assignGroup(1); // Assign to group Ctrl+1
```

#### Resource Tracking (`api.game.resources.*`)

```javascript
// Watch resources with callback
await api.game.resources.watch({
    regions: {
        gold: { x: 10, y: 10, width: 80, height: 20 },
        wood: { x: 100, y: 10, width: 80, height: 20 },
    },
    interval: 1000,
    onChange: (changes, current) => {
        console.log('Resources changed:', changes);
    },
});

// Wait for resources
await api.game.resources.waitFor({ gold: 500, wood: 200, timeout: 30000 });

// Get current resources
const resources = await api.game.resources.getCurrent();
```

#### Game Menu Automation (`api.game.menus.*`)

```javascript
// Build structure
await api.game.menus.build('barracks');

// Train units
await api.game.menus.train('footman', { count: 5, queue: 1 });

// Research upgrades
await api.game.menus.research('blacksmith', 'weaponry');

// Custom sequence
await api.game.menus.runSequence([
    { action: 'open', target: 'build' },
    { action: 'select', target: 'barracks' },
    { action: 'confirm' },
]);
```

#### Game State (`api.game.*`)

```javascript
await api.game.waitForEnabled('.build-button', true);
await api.game.waitForText('.resource-gold', '500');
await api.game.waitForPopup('.modal', true);
await api.game.waitForResources({ gold: 500, wood: 200 });
```

### Phase 4: Verification & Stability

**Action Verification:**

- Compares AXTree before/after actions
- Retries failed actions up to 3 times
- Confirms state changes

**Stuck Detection:**

- Tracks last progress step
- Detects no state change after 5 attempts
- Saves screenshot to `logs/stuck-{timestamp}.png`
- **Throws error** to halt automation

```javascript
await api.gameAgent.run('Build army', {
    stuckDetection: true,
    maxAttemptsWithoutChange: 5,
});
// Throws: "Agent stuck - no state change after 5 attempts"
```

## Usage Example

```javascript
import { api } from './api/index.js';

await api.init(page);

// Start autonomous game agent
const result = await api.gameAgent.run('Build barracks, train 5 footmen, then attack', {
    maxSteps: 50,
    stepDelay: 500,
    stuckDetection: true,
    maxAttemptsWithoutChange: 5,
});

console.log('Result:', result);
```
