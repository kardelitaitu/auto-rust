## What is a Task?

A **task** is a reusable automation script that performs a specific action in the browser. Tasks are:

- Loaded dynamically by name from CLI
- Executed within isolated browser sessions
- Expected to return structured results
- Integrated with circuit breaker and health monitoring

**Examples:**
- `pageview` - Visit and read a webpage
- `twitterFollow` - Follow a Twitter user
- `retweet` - Retweet a specific tweet
- `owb` - Autonomous agent task

## Task Execution Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                        main.js                                   │
│  node main.js taskName=url                                       │
└─────────────────────┬───────────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────────┐
│                    Orchestrator                                  │
│  1. Parse task name and payload                                  │
│  2. Validate payload against schema                              │
│  3. Discover browser and create session                          │
│  4. Load task module from tasks/{taskName}.js                    │
└─────────────────────┬───────────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────────┐
│                    Task Module                                   │
│  export default async function taskName(page, payload) {         │
│    // Task logic here                                            │
│    return createSuccessResult(...);                              │
│  }                                                               │
└─────────────────────┬───────────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────────┐
│                    Result Processing                             │
│  1. Capture structured result                                    │
│  2. Log to dashboard                                             │
│  3. Print summary to console                                     │
│  4. Exit with code 0 (success) or 1 (failure)                    │
└─────────────────────────────────────────────────────────────────┘
```

## Task Structure

Every task module must:

1. **Export a default async function**
2. **Accept `page` and `payload` parameters**
3. **Return a structured result**

### Function Signature

```javascript
/**
 * Task Description
 * @param {Page} page - Playwright page instance
 * @param {object} payload - Task payload with parameters
 * @returns {Promise<object>} Structured task result
 */
export default async function taskName(page, payload) {
    const startTime = Date.now();
    const browserInfo = payload.browserInfo || "unknown";
    const logger = createLogger("taskName.js");
    
    try {
        // Task logic here
        await api.withPage(page, async () => {
            await api.init(page);
            // ... automation
        });
        
        return createSuccessResult('taskName', {
            // Task-specific data
        }, { startTime, sessionId: browserInfo });
        
    } catch (error) {
        logger.error(`Task failed: ${error.message}`);
        return createFailedResult('taskName', error, {
            partialData: { /* any partial results */ },
            sessionId: browserInfo
        });
    }
}
```

## Returning Results

### Success Result

```javascript
import { createSuccessResult } from '../api/core/task-result.js';

return createSuccessResult('taskName', {
    // Task-specific data
    url: 'https://example.com',
    itemsProcessed: 5,
    actionTaken: 'follow'
}, {
    startTime: Date.now(),
    sessionId: 'browser-1'
});
```

**Result Structure:**
```javascript
{
  taskName: 'taskName',
  status: 'success',
  data: {
    url: 'https://example.com',
    itemsProcessed: 5
  },
  duration: 45.2,
  sessionId: 'browser-1'
}
```

### Failure Result

```javascript
import { createFailedResult } from '../api/core/task-result.js';

return createFailedResult('taskName', error, {
  partialData: {
    url: 'https://example.com',
    itemsProcessed: 2  // Partial progress
  },
  sessionId: 'browser-1'
});
```

**Result Structure:**
```javascript
{
  taskName: 'taskName',
  status: 'failed',
  error: {
    name: 'NavigationError',
    message: 'Timeout waiting for element',
    code: 'TIMEOUT',
    recoverable: true
  },
  partialData: {
    url: 'https://example.com'
  },
  duration: 12.5,
  sessionId: 'browser-1'
}
```

## Error Handling

### Catch and Return

```javascript
try {
    await api.goto(url);
} catch (error) {
    logger.error(`Navigation failed: ${error.message}`);
    return createFailedResult('taskName', error, {
        partialData: { url },
        sessionId: browserInfo
    });
}
```

### Validation Errors

```javascript
// Validate inputs
if (!payload.url) {
    return createFailedResult('taskName', 'URL is required', {
        sessionId: browserInfo
    });
}

if (!isValidUrl(payload.url)) {
    return createFailedResult('taskName', 'Invalid URL format', {
        sessionId: browserInfo
    });
}
```

### Timeout Handling

```javascript
const taskTimeoutMs = 60000;

await Promise.race([
    (async () => {
        // Task logic
    })(),
    new Promise((_, reject) =>
        setTimeout(() => 
            reject(new Error(`Task timeout (${taskTimeoutMs}ms)`)), 
            taskTimeoutMs
        )
    )
]);
```

## Example Task

### Simple Task: `tasks/hello.js`

```javascript
/**
 * Hello World Task
 * Demonstrates basic task structure
 * @module tasks/hello
 */

import { api } from "../api/index.js";
import { createLogger } from "../api/core/logger.js";
import { 
    createSuccessResult, 
    createFailedResult 
} from "../api/core/task-result.js";

/**
 * @param {Page} page - Playwright page
 * @param {object} payload - Task payload
 * @param {string} payload.url - URL to visit
 * @returns {Promise<object>} Task result
 */
export default async function hello(page, payload) {
    const startTime = Date.now();
    const browserInfo = payload.browserInfo || "unknown";
    const logger = createLogger("hello.js");
    
    logger.info("Starting hello task...");
    
    // Validate payload
    if (!payload.url) {
        return createFailedResult('hello', 'URL is required', {
            sessionId: browserInfo
        });
    }
    
    try {
        await api.withPage(page, async () => {
            // Initialize session
            await api.init(page, {
                persona: 'casual',
                logger
            });
            
            // Navigate to URL
            logger.info(`Navigating to ${payload.url}...`);
            await api.goto(payload.url, {
                waitUntil: 'domcontentloaded',
                timeout: 30000
            });
            
            // Get page title
            const title = await api.getText('title');
            logger.info(`Page title: ${title}`);
            
            // Take screenshot
            await api.screenshot.save('hello-task.png');
            
            // Return success
            return createSuccessResult('hello', {
                url: payload.url,
                title,
                screenshot: 'hello-task.png'
            }, {
                startTime,
                sessionId: browserInfo
            });
        });
        
    } catch (error) {
        logger.error(`Hello task failed: ${error.message}`);
        return createFailedResult('hello', error, {
            partialData: { 
                url: payload.url,
                title: null
            },
            sessionId: browserInfo
        });
    }
}
```

### Running the Task

```bash
# Basic usage
node main.js hello=url=https://example.com

# Output:
# ========== TASK RESULTS ==========
# Total: 1
# Success: 1
# Failed: 0
# Success Rate: 100.0%
# ✅ hello (45.2s)
# ==================================
```

## Best Practices

### 1. Always Return Structured Results

```javascript
// ✅ Good
return createSuccessResult('taskName', data, options);

// ❌ Bad
return;  // undefined result
return { success: true };  // unstructured
```

### 2. Validate Inputs Early

```javascript
// Validate at the start
if (!payload.url) {
    return createFailedResult('taskName', 'URL required');
}

// Then proceed with confidence
await api.goto(payload.url);
```

### 3. Use api.withPage()

```javascript
// ✅ Good - proper session isolation
await api.withPage(page, async () => {
    await api.init(page);
    // ... task logic
});

// ❌ Bad - no isolation
await api.init(page);
// ... task logic
```

### 4. Log Appropriately

```javascript
const logger = createLogger("taskName.js");

logger.info("Starting task...");      // Normal operation
logger.warn("Element not found");     // Recoverable issue
logger.error(`Failed: ${error.message}`);  // Task failure
```

### 5. Include Partial Data on Failure

```javascript
// Even on failure, return what was accomplished
return createFailedResult('taskName', error, {
    partialData: {
        url: payload.url,
        stepsCompleted: 3,
        lastSuccessfulAction: 'navigation'
    }
});
```

### 6. Handle Timeouts

```javascript
const timeout = 60000;

await Promise.race([
    doTask(),
    new Promise((_, reject) =>
        setTimeout(() => reject(new Error('Timeout')), timeout)
    )
]);
```

### 7. Clean Up Resources

```javascript
try {
    // Task logic
} finally {
    // Always close page
    if (page && !page.isClosed()) {
        await page.close();
    }
}
```

## Testing Tasks

### Unit Test Example

```javascript
// api/tests/unit/tasks-hello.test.js
import { describe, it, expect, vi, beforeEach } from 'vitest';
import helloTask from '../../tasks/hello.js';

describe('tasks/hello', () => {
    let mockPage;
    
    beforeEach(() => {
        mockPage = {
            goto: vi.fn(),
            close: vi.fn(),
            isClosed: vi.fn().mockReturnValue(false)
        };
    });
    
    it('should return success result', async () => {
        const result = await helloTask(mockPage, {
            url: 'https://example.com',
            browserInfo: 'test'
        });
        
        expect(result.status).toBe('success');
        expect(result.taskName).toBe('hello');
        expect(result.data.url).toBe('https://example.com');
    });
    
    it('should return failure for missing URL', async () => {
        const result = await helloTask(mockPage, {
            browserInfo: 'test'
        });
        
        expect(result.status).toBe('failed');
        expect(result.error.message).toContain('required');
    });
});
```

### Manual Testing

```bash
# Test with valid input
node main.js hello=url=https://example.com

# Test with invalid input (should fail gracefully)
node main.js hello=url=invalid-url

# Check exit code
echo $?  # Should be 0 for success, 1 for failure
```

## Task Payload Schema

Register your task schema for automatic validation:

```javascript
// api/utils/task-validator.js
const TASK_SCHEMAS = {
    hello: {
        required: {},
        optional: {
            url: 'url',
            timeout: 'number'
        },
        validate: (payload) => {
            const errors = [];
            if (payload.url && !isValidUrl(payload.url)) {
                errors.push('url must be a valid URL');
            }
            return errors;
        }
    }
};
```
