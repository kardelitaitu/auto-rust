# Tasks Specification

Current standards for creating or migrating tasks in this project.

## General Requirements

- **Parameters**: All task entry points (usually exported as `run`) should accept `(payload, page)`.
- **Payload**: Contains task-specific parameters and metadata like `browserInfo` (the session ID).
- **Timeouts**: Use `taskTimeoutMs` from the payload if available to bound task execution.

## Logging Standard

- **Initialization**: Use `createLogger('filename.js')`. Do not include session IDs or hardcoded task names in the logger initialization.
- **Message Format**: Keep log messages clean. The logger automatically prefixes messages with `[sessionId][taskName][scriptName]`. Avoid redundant prefixes like `[mytask] Message`.

## API & Context Standard

- **Context Preservation**: Always wrap the main task logic in `api.withPage(page, fn, options)`.
- **Options**: Pass `taskName` and `sessionId` in the options object to ensure all nested `api` calls (like `api.click`, `api.scroll`) have the correct context tags.

## Implementation Example

```javascript
import { createLogger } from '../api/core/logger.js';
import api from '../api/index.js';

/**
 * Example Task Implementation
 */
export async function run(payload, page) {
    const browserInfo = payload.browserInfo || 'unknown_profile';
    const logger = createLogger('exampleTask.js');

    logger.info('Initializing example task...');

    return await api.withPage(
        page,
        async () => {
            try {
                logger.info('Navigating to destination...');
                await api.goto('https://x.com/home');

                logger.info('Performing interactions...');
                await api.click('button[data-testid="tweetButton"]');

                return { success: true };
            } catch (error) {
                logger.error(`Task failed: ${error.message}`);
                return { success: false, error: error.message };
            }
        },
        {
            taskName: 'example',
            sessionId: browserInfo,
        }
    );
}
```
