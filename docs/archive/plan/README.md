# Plan Folder

This folder is archive-only now.
Active planning and proposals should go in `docs/specs/`.

## What Is Here

- `archive/`: completed, superseded, or historical TwitterActivity planning docs
- `twitterActivity/`: untouched legacy subfolder

## Reading Rule

- If a doc says `implemented`, `complete`, `history`, or `decision`, treat it as reference only.
- If a doc is still a proposal, convert it into a spec package before implementing.
